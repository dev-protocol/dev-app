# dev-app

This repository manages the resources of `devtoken.rocks`.

# Getting Started

How to install:

```bash
git clone git@github.com:frame00/dev-app.git
cd dev-app
npm i
```

# How to develop

Build TypeScript and run the local server.

```bash
npm start
```

Please check your code with TSLint.

```bash
npm run lint
```
